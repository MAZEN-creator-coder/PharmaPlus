1. SUPER ADMIN GLOBAL ANALYTICS
================================================================================

Purpose:
1. Display platform-wide statistics and analytics dashboard
2. Monitor pharmacy performance and revenue
3. Track user growth and order metrics

Technologies:
1. React Hooks (useState, useEffect)
2. Recharts - Charts visualization
3. Context API - Authentication
4. API Integration - /api/analytics-super

Key Components:
1. Performance Cards - KPI display
2. User Growth Chart (LineChart)
3. Revenue Distribution (PieChart)
4. Top Pharmacies (BarChart)
5. Pharmacy Performance Table
6. Search & Filter functionality
7. Export Reports

Data Displayed:
1. Total users with growth percentage
2. Total revenue (K format)
3. Active pharmacies count
4. Average order value
5. User growth trends
6. Revenue by categories
7. Pharmacy details and sales

Features:
1. Real-time data fetching
2. Interactive charts
3. Sorting & filtering
4. Export functionality
5. Loading spinner
6. Error handling
7. Responsive design

Access: Super Admin Only

2. SEARCH MEDICINE
================================================================================

Purpose:
1. Search medicines by name and category
2. Find nearby pharmacies
3. Compare prices across pharmacies

Technologies:
1. React Context (SearchMedicineContext)
2. Geolocation API
3. API Integration - /api/medicines/search
4. Leaflet - Map display

Key Components:
1. Search Bar with autocomplete
2. Location Filter
3. Price Range Filter
4. Search Results Grid
5. Medicine Card
6. Pharmacy Location Map
7. Sorting Options
8. Pagination

Features:
1. Real-time search
2. Advanced filtering
3. Price comparison
4. Map view
5. Quick add to cart
6. Wishlist/Favorites
7. Delivery time estimate

Data Displayed:
1. Medicine name and details
2. Active ingredient and strength
3. Price from each pharmacy
4. Pharmacy distance and rating
5. Stock availability
6. Delivery estimate

Access: All Users
